## 參數

{{< bs/config-toggle filename=params title="Site Parameters" >}}
hb:
  header:
    git_repo:
      service: github
      owner: hbstack
      name: hb
{{< /bs/config-toggle >}}

### `service`

Git 服務商，支持 `github`。

### `owner`

倉庫所有者。

### `name`

倉庫名稱。

## API 身份驗證和速率限制

### GitHub

你可能需要設置 `GITHUB_TOKEN` 環境變量以避免遇到 API 速率限制問題。
