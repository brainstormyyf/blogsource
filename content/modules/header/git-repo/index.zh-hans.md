## 参数

{{< bs/config-toggle filename=params title="Site Parameters" >}}
hb:
  header:
    git_repo:
      service: github
      owner: hbstack
      name: hb
{{< /bs/config-toggle >}}

### `service`

Git 服务商，支持 `github`。

### `owner`

仓库所有者。

### `name`

仓库名称。

## API 身份验证和速率限制

### GitHub

你可能需要设置 `GITHUB_TOKEN` 环境变量以避免遇到 API 速率限制问题。
