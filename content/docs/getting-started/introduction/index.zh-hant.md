## 動機

我創建和維護着一個功能豐富的主題 - [Hugo Bootstrap 主題](https://hbs.razonyang.com/)，但在開發之初，我沒有考慮過模塊化的問題，當開發新主題時，才意識到無法很好地複用已有的功能特性。於是乎，我着手於將該主題的功能特性模塊化，旨在使主題開發更加輕鬆。
